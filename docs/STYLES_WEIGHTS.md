# Numerical values for text styles and weights

## Styles

| Value  | Style |
| :---: | :--- |
| 1 | paragraph  |
| 2 | big header  |
| 3 | small header  |
| 4 | unordered list  |
| 5 | nested unordered list  |
| 6 | checkbox  |
| 7 | checked checkbox  |

## Weights

| Value  | Weight |
| :---: | :--- |
| 1 | normal  |
| 2 | bold  |
| 3 | italic  |
| 4 | bold italic  |
